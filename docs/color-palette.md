# Color Palette (Sprint 01)

## Brand Tokens

- `BrandPurple` `#6316F4`
- `BrandPurpleDark` `#3B0A94`
- `BrandPurpleLight` `#F1E8FF`
- `BrandYellow` `#FFCA28`
- `BrandSurface` `#FFFBF3`
- `BrandSurfaceDark` `#160D25`
- `BrandText` `#24163A`
- `BrandTextMuted` `#6A5D80`
- `BrandStroke` `#E4D9F9`

## Usage

- Primary accents: purple/yellow gradients on hero sections.
- Main surfaces: Material `surface` and `background`.
- Secondary blocks: `surfaceVariant`, `secondaryContainer`.
- Text hierarchy:
  - primary text on `onSurface`
  - secondary text on `onSurfaceVariant`

## Accessibility Notes

- Body text always uses Material on-color tokens for contrast.
- Large decorative gradients are paired with white text only when contrast is sufficient.
- Interactive controls rely on Material 3 defaults for state visibility.
